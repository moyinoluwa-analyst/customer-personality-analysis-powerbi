# Customer Personality Marketing Analysis (Power BI)

## Project Overview

This project analyzes customer demographics, spending behavior, and marketing campaign performance to help improve customer targeting and marketing effectiveness.

The dashboard was developed in Power BI and focuses on understanding customer characteristics, purchasing behavior, and campaign response patterns.



## Tools Used

* Power BI
* Power Query
* DAX
* Excel



## Business Objectives

* Understand customer demographics
* Analyze spending behavior
* Evaluate marketing campaign performance
* Identify high-value customer segments
* Support data-driven marketing decisions



## Data Preparation

The dataset was cleaned using Power Query.

### Cleaning Activities

* Checked for missing values
* Standardized data types
* Removed inconsistencies
* Created calculated columns
* Prepared measures for dashboard reporting



## Dashboard Components

### Executive Dashboard

Provides insights into:

* Total Customers
* Average Income
* Average Spending
* Customer Demographics
* Product Spending Categories

### Marketing Campaign Analysis

Provides insights into:

* Campaign Acceptance Rates
* Customer Response Behavior
* Segment Performance
* Marketing Effectiveness



## Key Findings

* Higher-income customers generally spent more across product categories.
* Certain customer segments responded significantly better to marketing campaigns.
* Wine and meat products represented major spending categories.
* Campaign acceptance rates varied considerably across customer groups.
* Customer demographics influenced purchasing behavior.



## Business Recommendations

* Focus marketing efforts on high-value customer segments.
* Personalize campaigns based on customer spending patterns.
* Improve targeting of underperforming campaigns.
* Develop retention strategies for high-spending customers.
* Increase investment in the most responsive customer segments.
